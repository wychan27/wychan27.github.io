Boolean and If-Statement Exercises

Create separate HTML file for these exercises.
Use the <script> tag

  1. Create a variable called 'hour' and save the current hour of the day (use the 24-hour format and save it as a number between 0 and 23).
    • If hour is between 6 and 12, display 'Good morning!' in the console
    • If hour is between 13 and 17, display 'Good afternoon!' in the console
    • Otherwise, display 'Good night1' in the console
    • Change the value in the 'hour' variable to make it display different messages.
  2. Continuing with Exercise 1 above, create variable called 'name and save your name inside (as a string).
    Update the if-statement to display your name in the message. For example: 'Good morning ${name}!'
  3. Imagine Harry Potter World in Florida has a discount for children (6 years and younger) or seniors (65 years and older).
    • Create a variable 'age' and save a person's age inside.
    • Create an if-statement that checks if the person qualifies for a discount.
    • If they do, display 'Discount' in the console
    • Otherwise, display 'No discount' in the console
    • Try changing the 'age' variable to display different messages
    Note: try to use the || operator in your solution
  4. Continuing from Exercise #3 (above), assume the discount is only available if it is NOT a holiday.
    • Create a variable: const isHoliday = true;
    • Update the code so that in order to get a discount, people must meet the age requirement AND it is also not a holiday.
    • Note: && has a higher priority than || so you may need to use brackets () to control which code gets done first
    • Try changing the value of isHoliday to display different messages
    Hints - but try it on your own first!

Coin-Flip Game

  1. Create a coin-flip.html file
  2. In the <script> tag, generate a random number with Math.random(); and save it in a variable
  3. Create an if-statement and check:
    • if the number is less than 0.5, then display 'heads' in the console
    • Else display 'tails' in the console
    Hint - but try it on your own first!
  4. Instead of displaying 'heads' or 'tails' in the console, save the result in a variable called, 'result'
  5. Let's say we are trying to guess the result. Create a variable called, 'guess' and save your guess ('heads' or 'tails').
    • If your guess matches the result, display 'You win!' in the console
    • If your guess does not match the result, display 'You lose!' in the console
    Hint - but try it on your own first!
