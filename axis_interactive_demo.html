üéì What You're Learning:

The Key Insight: justify-content and align-items don't control "horizontal" and "vertical" - they control the main axis and cross axis!

  • Main Axis: The direction items flow (set by flex-direction)
  • Cross Axis: Perpendicular to the main axis
  • justify-content: Always positions along the main axis
  • align-items: Always positions along the cross axis

Try This: Change flex-direction and watch how the same justify-content values produce completely different results!
